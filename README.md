# PUT versus PATCH

This is a companion repository to go with my blog post, PUT versus PATCH. The API is far from perfect, but hopefully it's enough to illusrate the point!

## Getting started

1. Run `npm install` to install the dependencies listed in `package.json`.
2. Run `npm start` to start the server on port 3000.
3. Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to stop the server.
